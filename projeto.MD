Vamos criar projeto onde deve ser um builder de landing page.

Os elementos que poderão ser utilizados na landing page serão componentes pre-criados (componentes react) que serao costumizaveis atraves de props.

Estas landing pages na maioria das vezes serão apenas uma landing page de um produto com variants e uma url que será o redirecionar o usuário para algum checkout externo

Cada landing page ficara em um subdomain separado, duas landing pages nao poderao ser cadastradas em um mesmo subdomain

A configuração do apontamento do subdominio sera feito pela equipe de infra.

Para construir e gerenciar as landing pages, sera uma visao admin, e para montar a landing page pensei em utilizar a lib pluck do react. Mas pode utilizar uma solucao melhor caso exista

Por enquanto vamos tratar isto como um MVP
